# ChatApp
###### Realtime chat application where users can have one-to-one and group chats. the user will get notified when somebody posts a message. Used Socket.io to establish bi-directional communication between clients and servers. having features like typing indicator, user profile with image, search bar with a side drawer having your chats, and adding and removing users in a group chat by the group admin. Technologies used are React.js, Javascript, MongoDB, Node.js, Express.js.
